## Sobre o repositório

Este repositório centraliza os exemplos práticos trabalhados ao longo das aulas. Cada diretório corresponde a um conteúdo específico do curso, contendo o código completo e funcional desenvolvido em sala. O material foi estruturado para servir tanto como referência durante o curso quanto como base de consulta após sua conclusão.

## Tecnologias abordadas

- **HTML5** — estruturação semântica de páginas web
- **CSS3** — estilização, layout responsivo e animações
- **JavaScript (ES6+)** — interatividade e lógica de programação no cliente
- **Git e GitHub** — versionamento e colaboração

## Como executar os projetos

Para executar qualquer um dos projetos localmente:

```bash
# Clone o repositório
git clone https://github.com/mariannarossim/senac-programacao-web.git

# Acesse o diretório do projeto desejado
cd senac-programacao-web/to-do-list

# Abra o arquivo index.html no navegador
```

Recomenda-se o uso da extensão **Live Server** no Visual Studio Code para execução com recarregamento automático durante a prática.

## Estrutura do repositório

Cada projeto é organizado em diretórios próprios, separando os arquivos por tipo (HTML, CSS e JavaScript), conforme boas práticas de organização de projetos web.

```
senac-programacao-web/
├── README.md
└── to-do-list/
    ├── index.html         # página inicial (landing)
    ├── README.md
    ├── css/
    │   ├── index.css
    │   └── list.css
    ├── html/
    │   └── list.html      # página da lista de tarefas
    └── js/
        └── list.js        # lógica da lista
```

## Objetivos de aprendizagem

Ao final do curso, espera-se que o aluno seja capaz de:

- Compreender e aplicar os fundamentos do desenvolvimento web front-end
- Estruturar páginas HTML semânticas e acessíveis
- Desenvolver interfaces responsivas e modernas com CSS
- Implementar interatividade e lógica de programação com JavaScript
- Versionar projetos utilizando Git e GitHub
- Aplicar boas práticas de organização e código limpo
